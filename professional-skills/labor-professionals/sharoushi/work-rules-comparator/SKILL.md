# 就業規則比較分析システム

## スキル概要
自社の就業規則を法令や他社事例と比較し、改善点を提示する就業規則の最適化支援システムです。

## 開発者情報
- **専門家**: 中村美和
- **資格**: 社会保険労務士
- **専門分野**: 社会保険・労働保険

## 主な機能

### 1. 法令適合性チェック
- 労働基準法との整合性確認
- 最新法改正への対応状況診断
- 必須記載事項の網羅性確認
- リスク条項の自動検出

### 2. 業界比較分析
- 同業他社との条件比較
- 業界標準からの乖離分析
- ベストプラクティスの提案
- 競争力評価レポート

### 3. 改善提案機能
- 問題条項の具体的修正案
- 優先度別の改善リスト
- 導入効果のシミュレーション
- 改定スケジュールの作成

### 4. 従業員影響分析
- 変更による影響度評価
- 従業員説明資料の自動生成
- FAQ の作成支援
- 同意取得プロセス管理

## ワークフロー

1. **現状分析**
   - 現行規則のアップロード
   - 自動テキスト解析
   - 問題点の抽出

2. **比較検証**
   - 法令チェックの実施
   - 業界データとの比較
   - ギャップ分析

3. **改善計画**
   - 修正案の作成
   - 影響範囲の特定
   - 実施計画の策定

4. **実装支援**
   - 改定案の最終化
   - 従業員周知の実施
   - 効果測定の設定
