# 宮川行政書士事務所 - ビジネス分析メモ

## 企業概要
- **会社名**: 宮川譲行政書士事務所
- **業種**: 専門サービス業（行政書士事務所）
- **所在地**: 香川県高松市
- **規模**: 7名（2013年創業時1名から成長）
- **理念**: 100年先も地域と共に歩む

## 主要サービス
1. **建設業関連**
   - 建設業許可申請（新規・更新）
   - 経営事項審査
   - 入札参加資格申請

2. **相続・承継関連**
   - 相続対策・認知症対策
   - 遺言書作成
   - 事業承継・M&A
   - 民事信託

3. **各種許認可**
   - 産業廃棄物収集運搬業許可
   - 法人設立（医療法人、NPO、宗教法人）
   - 各種営業許可（古物商、旅館業、酒類販売等）

4. **国際業務**
   - 永住許可申請
   - 経営管理ビザ
   - 帰化申請
   - 在留資格相談

## 現在の課題・ペイン
1. **業務効率化の課題**
   - 複雑な行政手続きの管理
   - 長期間（最長1年）にわたる案件進捗管理
   - 多種多様な書類作成・管理

2. **顧客対応の課題**
   - 案件ステータスの問い合わせ対応
   - 期限管理とリマインダー
   - 顧客への進捗報告

3. **情報管理の課題**
   - 法改正・制度変更への対応
   - 顧客情報・書類の一元管理
   - セキュアな文書共有

4. **スケーラビリティの課題**
   - 人員増加に伴う業務標準化
   - ナレッジ共有の仕組み
   - 新人教育の効率化

## デジタル化ニーズ
- オンライン予約システム
- 顧客ポータル（書類提出・進捗確認）
- 業務管理システム（CRM）
- 文書自動生成システム
- 多言語対応（国際業務向け）
- 電子決済対応
